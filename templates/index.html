Site para programadores.

Linguagens de programação:

  • Linguagem de programação Python.
    É uma linguagem Open-Source de propósito geral usado bastante em data science, machine learning, desenvolvimento de web, desenvolvimento de aplicativos, automação de scripts, fintechs e mais.

  • Linguagem de programação Java.
    É possível criar aplicativos para dispositivos móveis, tais como smartphones e tablets. Em uma época em que tais dispositivos são comuns, essa linguagem ganha muita relevância.

  • Linguagem de programação NodeJS.
    Software de código aberto, multiplataforma, baseado no interpretador V8 do Google e que permite a execução de códigos JavaScript fora de um navegador web.

Sistemas operacionais:

  • Ubuntu
    Ubuntu é um sistema operacional ou sistema operativo de código aberto, construído a partir do núcleo Linux, baseado no Debian e utiliza GNOME como ambiente de desktop de sua mais recente versão com suporte de longo prazo. Esta distribuição Linux é desenvolvida pela Canonical Ltd.

  • Debian
    Debian, anteriormente chamado Debian GNU/Linux e hoje apenas Debian, é um sistema operacional composto inteiramente de software livre e mantido oficialmente pelo Projeto Debian. O projeto recebe, ainda, apoio de outros indivíduos e organizações de todo o mundo.

  • Windows
    Microsoft Windows é uma série popular de sistema operacional criada pela empresa Microsoft fundada por Bill Gates e Paul Allen. O Windows é um produto comercial e o preço de cada versão é diferente. Apesar de instalar um grande número de cópias ilegais, é o sistema operacional mais usado no mundo.
